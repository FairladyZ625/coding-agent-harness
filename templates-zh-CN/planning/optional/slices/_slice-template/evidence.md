# 切片证据

| 证据 ID | 类型 | 路径 | 摘要 | 状态 |
| --- | --- | --- | --- | --- |
| E-001 | command / diff / fixture / screenshot / review / report | PUBLIC:path 或 PRIVATE:path 或 TARGET:path 或 EXTERNAL:path 或 URL:https://example.com | [这份证据证明了什么] | missing |

## 状态说明

- `missing`：证据尚未收集。
- `partial`：已有部分证据，但不足以关闭切片。
- `present`：证据完整，可复查。
- `waived`：证据被明确豁免，并有原因。
