# Coding Agent Harness

Coding Agent Harness es una capa de trabajo nativa del repositorio para agentes de programacion. Ayuda a Codex, Claude Code, Gemini CLI, agentes estilo Cursor y herramientas similares a planificar, revisar, verificar y retomar trabajo de software de larga duracion.

![Dashboard overview](../assets/dashboard-overview.png)

## Como funciona

1. Instala el Skill o ejecuta la CLI con `npx`.
2. El agente diagnostica el repositorio y propone un plan de inicializacion o migracion.
3. El Dashboard local muestra tareas, advertencias, evidencia y estado de revision.
4. Ejecuta las verificaciones de harness antes de entregar.
